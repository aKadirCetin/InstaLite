# InstaLite

InstaLite is a lightweight social media Android application inspired by Instagram.  
The project was developed using Java and Firebase services.

## Features

- User authentication system
- Create and share posts
- Upload and display images
- Real-time post feed
- Firebase cloud integration
- Simple and modern UI

## Tech Stack

- Java
- Android SDK
- Firebase Authentication
- Firebase Firestore
- Firebase Storage
- Firebase Analytics
- Firebase App Check
- RecyclerView
- Picasso
